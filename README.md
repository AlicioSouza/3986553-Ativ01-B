# 3986553-Ativ01B

contador de pessoas com visor LCD

## Autor
Alicio Rodrigues de Souza

## RU
3986553

## Descrição
Projeto desenvolvido utilizando o microcontrolador PIC16F877A para controle de entrada e saída de pessoas com exibição em display LCD 16x2.

## Hardware utilizado
- PIC16F877A
- LCD 16x2
- Sensores LDR
- LEDs de indicação
- Placa PIC-DIP40
- PICkit 3.5
